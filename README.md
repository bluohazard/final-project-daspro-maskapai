# Maskapai
Simple java application for purchasing airplane tickets

## Installation

1. Clone the repo
```
git clone https://github.com/bluohazard/final-project-daspro-maskapai.git
```
<p align="center">
  <img src="gif/git-clone.gif" alt="Git Clone">
</p>

2. Compile this project
```
javac maskapai.java
```
3. Run this project
```
java maskapai
```
